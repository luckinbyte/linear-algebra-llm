# 第三部分：拆解 Transformer

终于到了最激动人心的部分。

在前面的章节中，我们积累了足够的数学工具。现在，我们要把它们全部用上——**逐层拆解 Transformer**，这个驱动了 GPT、BERT、Llama 等所有现代大语言模型的核心架构。

你将会看到：

- **张量运算**如何让 GPU 同时处理上千个请求
- **注意力机制**本质上就是一组精心设计的矩阵乘法
- **位置编码**用巧妙的数学让模型理解"顺序"
- **Softmax**把任意数字变成优雅的概率分布

读完这一部分，当你再看到 Transformer 的论文公式时，不会再感到畏惧——因为每一个符号，你都知道它从哪里来、为什么在那里。
